# OpenAI API 密钥获取和配置指南

## 🔑 如何获取 OpenAI API 密钥

### 方法1：官方 OpenAI（推荐）

1. **访问 OpenAI 官网**：
   - 打开 https://platform.openai.com
   - 点击右上角 "Sign up" 或 "Log in"

2. **创建账户**：
   - 使用邮箱注册新账户
   - 验证邮箱地址
   - 完成手机号验证

3. **获取 API 密钥**：
   - 登录后，点击右上角头像
   - 选择 "View API keys"
   - 点击 "Create new secret key"
   - 复制生成的密钥（格式：sk-xxxxxxxx）

4. **充值账户**：
   - OpenAI 需要预付费使用
   - 最低充值 $5 USD
   - 支持信用卡付款

### 方法2：国内代理服务（更方便）

如果无法直接使用 OpenAI，可以考虑国内的代理服务：

1. **ChatAI**：https://chatai.com
2. **API2D**：https://api2d.com  
3. **CloseAI**：https://close-ai.com

这些服务通常：
- 支持支付宝/微信付款
- 提供相同的 API 接口
- 价格相对便宜

## 🚀 在应用中配置 API 密钥

### 方法1：在PDF界面直接配置（推荐）

1. 打开应用程序
2. 点击 "PDF演讲稿" 标签页
3. 在 "API密钥" 输入框中粘贴您的密钥
4. 点击 "保存密钥" 按钮
5. 看到成功提示即可使用

### 方法2：在设置界面配置

1. 点击左下角 "设置" 按钮
2. 找到 "LLM大模型" 部分
3. 选择 "OpenAI" 服务
4. 在 "API Key" 字段输入密钥
5. 点击保存

## 🔧 测试配置

配置完成后，您可以：

1. 上传一个简单的PDF文件
2. 点击 "开始生成"
3. 观察是否能成功调用AI生成演讲稿

如果成功，您会看到：

## 🛠 故障排除

### 常见问题：

1. **"未配置API密钥"**：
   - 检查密钥是否以 `sk-` 开头
   - 确认已点击保存按钮
   - 重启应用程序

2. **"API调用失败"**：
   - 检查网络连接
   - 确认API密钥有效
   - 检查账户余额

3. **"密钥格式错误"**：
   - OpenAI密钥格式：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - 长度通常为51个字符
