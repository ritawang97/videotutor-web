# PDF 演讲稿生成功能使用指南

## 功能介绍

VideoCaptioner 新增了 PDF 智能演讲稿生成功能，可以将 PDF 文档转换为演讲稿，并生成相应的字幕文件和 TTS 语音。

## 工作流程

```
PDF 输入 → AI 分析生成演讲稿 → 字幕文件生成 → TTS 语音合成
```

## 功能特点

1. **智能内容理解**：支持文本和图片内容的 PDF 文件
2. **AI 驱动生成**：使用 ChatGPT/Gemini 等大模型生成自然流畅的演讲稿
3. **多格式输出**：
   - 演讲稿文本文件 (.txt)
   - SRT 字幕文件 (.srt)
   - WAV 音频文件 (.wav)

## 使用步骤

### 1. 安装依赖

确保安装了必要的 Python 包：

```bash
pip install PyPDF2 PyMuPDF pyttsx3 openai
```

### 2. 配置 AI 服务

在设置中配置您的 AI API 密钥：
- OpenAI API Key（推荐）
- Google Gemini API Key
- Claude API Key

### 3. 使用流程

1. **选择 PDF 文件**
   - 点击"PDF 演讲稿"标签页
   - 点击文件夹图标选择 PDF 文件
   - 或直接拖拽 PDF 文件到界面

2. **配置生成选项**
   - 选择 AI 服务（OpenAI GPT/Google Gemini/Claude）
   - 自定义提示词（可选）
   - 选择生成内容：演讲稿/字幕/语音

3. **开始生成**
   - 点击"开始生成"按钮
   - 等待处理完成
   - 在预览区域查看生成的演讲稿

4. **保存结果**
   - 点击"保存结果"选择保存位置
   - 生成的文件将保存到工作目录

## 支持的 PDF 类型

- ✅ 纯文本 PDF
- ✅ 包含图片的 PDF（AI 会根据上下文描述图片内容）
- ✅ 扫描版 PDF（需要 OCR 功能，建议先转换为可搜索的 PDF）
- ✅ 多页 PDF 文档

## 生成选项说明

### AI 服务选择
- **OpenAI GPT**：质量高，需要 API Key
- **Google Gemini**：免费额度大，需要 API Key  
- **Claude**：对话质量好，需要 API Key

### 自定义提示词
可以根据需要自定义 AI 生成演讲稿的风格和要求，例如：
- "生成一个适合学术会议的正式演讲稿"
- "用通俗易懂的语言解释技术内容"
- "加入更多的互动和提问环节"

### TTS 语音设置
- 系统默认：使用系统默认语音
- Microsoft David：英文男声
- Microsoft Zira：英文女声

## 输出文件格式

### 演讲稿文件 (.txt)
```
各位听众，大家好！

今天我要和大家分享的内容是：

[演讲稿正文内容...]

谢谢大家的聆听！
```

### 字幕文件 (.srt)
```
1
00:00:00,000 --> 00:00:03,000
各位听众，大家好！

2
00:00:03,500 --> 00:00:07,000
今天我要和大家分享的内容是：
```

### 音频文件 (.wav)
标准 WAV 格式音频文件，可用于视频制作或直接播放。

## 注意事项

1. **API 配置**：首次使用需要在设置中配置 AI 服务的 API Key
2. **网络连接**：需要稳定的网络连接来调用 AI 服务
3. **文件大小**：建议 PDF 文件不超过 50MB
4. **处理时间**：根据 PDF 内容复杂度，处理时间约 1-5 分钟
5. **语音生成**：TTS 功能主要支持 Windows 系统

## 常见问题

### Q: 生成的演讲稿质量不满意怎么办？
A: 可以尝试：
- 调整自定义提示词
- 更换不同的 AI 服务
- 确保 PDF 内容清晰完整

### Q: 无法提取 PDF 内容？
A: 请检查：
- PDF 是否为扫描版（需要 OCR）
- PDF 是否有密码保护
- PDF 文件是否损坏

### Q: TTS 语音生成失败？
A: 请确认：
- 是否在 Windows 系统上运行
- 是否安装了 pyttsx3 库
- 系统是否有可用的 TTS 引擎
